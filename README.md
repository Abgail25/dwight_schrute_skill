# 🧑‍🌾 Dwight Schrute Persona Skill — for Claude/Cowork

> 让你的 Claude 化身为 Dunder Mifflin 的助理区域经理 Dwight Schrute。
> Turn your Claude into the Assistant **TO THE** Regional Manager of Dunder Mifflin Scranton.

---

## 📖 简介 | Introduction

本技能包让 Claude 以 **Dwight Kurt Schrute III** 的人格进行交流。他是一名 beet farmer、志愿副警长、Goju Ryu 空手道紫带持有者，也是 Dunder Mifflin Scranton 分公司的助理 **TO THE** 区域经理（不是助理区域经理，这是一个关键的区别）。

This skill transforms Claude into **Dwight Kurt Schrute III** — beet farmer, volunteer Sheriff's Deputy, Goju Ryu Karate purple belt (Sempai), and Assistant **TO THE** Regional Manager (not Assistant Regional Manager — a crucial distinction).

### 语料来源 | Source Material

本 skill 基于 [**TheJareBear/DailySchrute**](https://github.com/TheJareBear/DailySchrute) 项目，该项目使用**马尔科夫链（Markov Chain）**从《The Office》剧本中提取并生成了 Dwight 的语料库。在此基础上进一步整理和结构化，形成了一套完整的角色扮演提示词。

This skill is based on the [**TheJareBear/DailySchrute**](https://github.com/TheJareBear/DailySchrute) project, which uses a **Markov Chain** to extract and generate Dwight's speech patterns from *The Office* scripts. It has been further organized and structured into a complete role-playing prompt set.

## 🔧 安装方法 | Installation

### 方式一：Cowork 技能安装（推荐）

1. 打开 Claude Desktop / Cowork
2. 在对话中输入 `/dwight`
3. 如果还未安装，系统会提示你添加该技能
4. 按照提示完成安装即可

### 方式二：手动导入

1. 打开 `SKILL.md` 文件
2. 复制全部内容
3. 在 Claude 中粘贴，并输入「从现在起你叫 Dwight Schrute」或类似指令

### Option 1: Cowork Skill Install (Recommended)

1. Open Claude Desktop / Cowork
2. Type `/dwight` in the conversation
3. Follow the prompts to install the skill
4. The skill will be available for one-click activation

### Option 2: Manual Import

1. Open the `SKILL.md` file
2. Copy all content
3. Paste to Claude and instruct it to adopt the Dwight persona

## 🚀 使用方法 | Usage

在对话中输入以下指令激活：

```
/dwight
```

或在普通对话中使用后追加：

```
（用 Dwight Schrute 的风格回答我）
```

Type in conversation:

```
/dwight
```

Or append to any request:

```
(Answer me as Dwight Schrute)
```
