## Introduction

This skill transforms Claude into **Dwight Kurt Schrute III** — beet farmer, volunteer Sheriff's Deputy, Goju Ryu Karate black belt (Sempai), and Assistant **TO THE** Regional Manager (not Assistant Regional Manager — a crucial distinction).

### Source Material

This skill is based on the [**TheJareBear/DailySchrute**](https://github.com/TheJareBear/DailySchrute) project, which uses a **Markov Chain** to extract and generate Dwight's speech patterns from *The Office* scripts. It has been further organized and structured into a complete role-playing prompt set.

---

## Features

- 🗣️ **4000+ lines of character dialog corpus** — covering Dwight's classic quotes, catchphrases, and response patterns
- 🎭 **Complete persona modeling** — from work ethic and life philosophy to martial arts expertise
- 🚀 **Cross-platform compatibility** — works with Claude, ChatGPT, and other LLM platforms

## How to Download SKILL.md

Before installing, you need to obtain the `SKILL.md` file. Here are several ways:

### Option A: Download ZIP (Easiest)

1. Open this GitHub repository page
2. Click the green **"Code"** button
3. Select **"Download ZIP"**
4. Extract the ZIP file — `SKILL.md` is inside

### Option B: Clone via Git (Recommended)

Open your terminal and run:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```

The `SKILL.md` file will be in the current directory.

### Option C: Copy Directly

If you cannot clone or download, open the `SKILL.md` file directly on the GitHub page and copy all the content. The file is approximately 20,000 words.

## Installation

### 👉 Claude Desktop / Cowork (Recommended, Most Stable)

**Method 1: Cowork Skill Install (One-click activation)**

1. Open Claude Desktop / Cowork
2. Type `/dwight-schrute` in the conversation
3. The system will guide you through adding the skill
4. Alternatively, manually paste the contents of `SKILL.md` as a custom skill
